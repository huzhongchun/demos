组件名称：刮刮卡组件

点击展开详细使用说明

说明：

(string) element: 需要添加刮刮卡的div类名或者id

(string) width: 图片的宽度

(string) height: 图片的高度

(string) img:需要显示的图片即中奖的图片

(string) backgroundImg: 背景图片例如灰色的图片

HTML
无
CSS样式
无
Javascript调用
var scratchcard = Jumei.getModule('ui.scratchcard'); new scratchcard({ element:'.card-area', width: 100, height: 20, img: 'http://p0.jmstatic.com/mobile/act/active301/702luodi/prize-area-un.jpg', backgroundImg: 'http://p0.jmstatic.com/mobile/act/active301/702luodi/prize-area.png' });
