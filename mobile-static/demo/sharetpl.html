组件名称：微博微信分享

点我你就知道了
点击展开详细使用说明

说明：

（string）sinaBtnBg: 新浪分享按钮图片的链接 ，默认：http://images.jumei.com/mobile/act/image/widget/share/weibo-btn.png

（string）weichatBtnBg : 微信分享给好友按钮图片的链接 ，默认：http://images.jumei.com/mobile/act/image/widget/share/wechat-btn.png

（string）weiFriendBtnBg : 微信分享到朋友圈按钮图片的链接 ，默认：http://images.jumei.com/mobile/act/image/widget/share/weixin_friend.png

（string）sinaLinkUrl: 分享的链接，默认本页面的url

（string）weiXinChatLinkUrl: 微信分享给好友的链接，默认本页面的url

（string）weiXinFriendLinkUrl: 微信分享到朋友圈的链接，默认本页面的url

（string）sinaPic: 新浪分享出去的图片的链接，默认：http://p0.jmstatic.com/templates/jumei/images/logo_new_v1.jpg

（string）weiXinChatPic: 微信分享给好友的图片的链接，默认：http://p0.jmstatic.com/templates/jumei/images/logo_new_v1.jpg

（string）weiXinFriendPic: 微信分享到朋友圈的图片的链接，默认：http://p0.jmstatic.com/templates/jumei/images/logo_new_v1.jpg

（string）sinaTitle: 新浪分享的文案，默认：精彩购物，尽在聚美优品

（string）weixinChatTitle: 微信分享给好友的标题文案，默认：'精彩购物，尽在聚美优品',

（string）weiXinFriendTitle: 微信分享到朋友圈分享的标题文案，默认：'精彩购物，尽在聚美优品',

（string）weiXinFriendContent: 微信分享给好友的正文文案，默认：’中国最大正品化妆品团购网站，千万用户推荐，拆封30天无条件退款！‘

（string）weiXinChatContent: 微信分享到朋友圈的正文文案，默认：’中国最大正品化妆品团购网站，千万用户推荐，拆封30天无条件退款！‘

注意：

避免id重名，如果分享框的最外层元素的id名（share-module），在你的页面之前已经定义过，则组件不会生效，提示你修改id名。

HTML
<!--HTML--> <div id="share-btn">点我你就知道了</div>
CSS样式
<link rel="stylesheet" type="text/css" href="http://ms0.jmstatic.com/lib/css/widget/share.css">
Javascript调用
var link_url = 'http://www.baidu.com'; var share_pic = 'http://p0.jmstatic.com/mobile/act/activities/2014_11/11sharepic/share.png'; var share = Jumei.getModule('ui.share'); var shareObj = new share({ sinaTitle: '#聚美11.11购物狂欢节#  不止5折，大牌减才是真的减，国际知名品牌尽在聚美优品！', weixinChatTitle: '聚美11.11购物狂欢节', weiXinFriendTitle: '聚美11.11购物狂欢节', sinaLinkUrl: link_url, weiXinChatLinkUrl: link_url, weiXinFriendLinkUrl: link_url, sinaPic: share_pic, weiXinChatPic: share_pic, weiXinFriendPic: share_pic, weiXinFriendContent:"不止5折，大牌减才是真的减，国际知名品牌尽在聚美优品！", weiXinChatContent:"不止5折，大牌减才是真的减，国际知名品牌尽在聚美优品！" }) $('#share-btn').on('click', function(){ shareObj.show(); });
