# 示例

## 使用示例

示例包括以下目录结构

* root
* * release
* * source 
* * logo.png
* * README.md

`release` 目录下为发布版本的示例。
`source`  目录下为项目源码。是一个完整的项目

** 导入项目 ** 

使用 EgretWing 选择导入->Egret 项目：导入`source`文件夹下的项目。

** 编译项目 **

使用 EgretWing 项目编译引擎按钮来编译引擎。

** 运行项目 **

使用构建并运行项目来运行项目。


## 示例详情

该示例中演示了如何初始化Egret3D引擎，如何创建一个目标摄像机，如何创建天空盒子。在示例中按键盘 W、S、A、D

可分别对像机进行上、下、左、右移动，点击鼠标左键，可旋转像机。
